Gestor de Tareas con Python y Flask

¿Qué es este proyecto?

Este es un gestor de tareas simple pero funcional, desarrollado como parte del aprendizaje de desarrollo web con Python y Flask. Permite crear, gestionar y dar seguimiento a tus tareas diarias de manera eficiente.

Tecnologías utilizadas

  • Python - Lenguaje de programación principal
  • Flask - Framework web
  • Bootstrap - Framework CSS
  • Font Awesome - Iconos

Funcionalidades

  • Crear nuevas tareas con título y descripción
  • Marcar tareas como completadas
  • Eliminar tareas
  • Interfaz responsive y amigable
  • Mensajes de retroalimentación al usuario
